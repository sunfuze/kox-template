# kox template
This is [kox](https://github.com/sunfuze/kox) template code
## Usage

```
 // install
 $ npm i -g vue-cli
 $ vue init sunfuze/kox-template my-project
 $ cd my-project
 $ npm i -g yarn
 $ yarn
 $ node app
```

server will listening to `3666`, you can access swagger document by `http://localhost:3666/documentation`

## Test

```
 $ npm run test
```
